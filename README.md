# java_hello_world

This is a simple java application.

## Dependencies
- make
- bash
- docker

**if you are using windows, you will have to use wsl**

## Make commands

---
### test
test application:
```bash
make test
```
---
### build
build executable fat jar:
```bash
make build_jar
```
---
### run
run:
```bash
make run
```
---
### dev_docker
open and enter a dev_docker container:
```bash
make dev_docker
```
